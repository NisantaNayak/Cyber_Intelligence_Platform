# CIP Frontend (cip-frontend)

React + Vite UI for the Cyber Intelligence Platform. Talks to the
**cip-backend** repo over tRPC.

## Type safety across repos

The backend's `AppRouter` type is **vendored** into `src/api-types.d.ts`
(committed). It is generated by the backend and copied here — the frontend
needs **zero backend dependencies**.

Regenerate after backend API changes:

```bash
# 1. in cip-backend/
npm run build:api-types        # emits cip-backend/dist/api-types.d.ts

# 2. in cip-frontend/
npm run sync:api-types         # copies it to src/api-types.d.ts (commit this)
```

`sync:api-types` assumes `cip-backend` is a sibling directory. Override with
`CIP_BACKEND_DIR=/path/to/cip-backend npm run sync:api-types`.

## Run

```bash
npm install
npm run sync:api-types         # if src/api-types.d.ts is missing/stale
npm run dev                    # http://localhost:5173  (expects API on :4000)
```

The API base URL and bearer token are in `src/trpc.ts`.
